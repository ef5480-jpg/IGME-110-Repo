# Minecraft Modding Listicle
## Useful Coding & Art/Sound Links
- **Coding**
  - [Fabric Wiki](https://wiki.fabricmc.net/start) - Fairly useful documentation of the Fabric loader
  - [Linkie](https://linkie.shedaniel.dev/) - Can view and convert mappings
  - [Modding by Kaupenjoe](https://www.youtube.com/@ModdingByKaupenjoe/videos) - Extremely useful tutorials for both java and modding
- **Art/Sound**
  - [Blockbench Style Guide](https://www.blockbench.net/wiki/guides/minecraft-style-guide/) - Useful for replicating the Minecraft style of pixel art and models
  - [Pixabay](https://pixabay.com/sound-effects/) - Lots of free sounds to use
  - [Audacity](https://www.audacityteam.org/) - A way to edit sounds
